### Backend

###To run 
Download the zip file then install node modules npm install and on terminal type node index.js or nodemon index.js

###Technologies used
NodeJS, MongoDB, Express, Multer, Bcrypt, Nodemailer, Json web token, mongoose,dotenv,cors
